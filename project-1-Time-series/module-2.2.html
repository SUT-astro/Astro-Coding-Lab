Moving-Average Model(MA Model)

Moving-Average Model เป็นหนึ่งในแนวทางการวิเคราะห์ข้อมูลที่เกี่ยวข้องกับแนวโน้มของข้อมูลในโดเมนเวลา ซึ่งใช้กันทั่วไปในการประมวลผล ทางเศรษฐมิติ และทางวิทยาศาสตร์ วิศวกรรมศาสตร์ต่างๆ โมเดลเหล่านี้มีเอกลักษณ์เฉพาะด้วยการใช้ฟังก์ชันตอบสนอง(response function) ซึ่งอธิบายว่าค่าในอดีตของโดเมนเวลาใดๆทั้งหมด ส่งผลค่าปัจจุบันอย่างไร ซึ่งแตกต่างจากโมเดล AR ที่จะกำหนดว่าเราจะพิจาณาจำนวนข้อมูลในอดีตเท่าไรก่อนจะนำมาคำนวนค่าในปัจจุบัน

ในโมเดล MA นั้นค่าปัจจุบันของโดเมนเวลาจะได้มาจากการ convolving( ด้วย convolution theorem) ที่ค่าผ่านมาของโดเมนเวลาใดๆด้วยฟังก์ชันตอบสนอง เพื่อให้ได้ค่าในปัจจุบันตามสมการ :

$$ x_t = \sum_{k=-\infty}^{\infty} C_k x_{t-k} $$

เมื่อ :

  • \(x_t\): เส้นโค้งแสง ณ เวลา t ใดๆ
  • \(C_k\): ฟังก์ชันการตอบสนอง(response function)หรือค่าสัมประสิทธิ์ตัวกรอง โดยพิจารณาการมีส่วนร่วมของค่าในอดีตแต่ละค่ากับค่าปัจจุบันของโดเมนเวลาตามฟังก์ชันการตอบสนองใดๆ
  • \(x_{t-k}\): เป็นพจน์ที่แสดงถึง ค่าที่ผ่านมาของโดเมนเวลาจะขึ้นอยู่กับ \(lag k\) โดยที่ \(k\) มีค่าตั้งแต่ \(-\infty\) ไปจนถึง \(-\infty\)

ตัวอย่างของฟังก์ชันโค้ดโมเดล MA ที่ใช้ในการการสร้างกราฟแสงจะเป็นดังนี้ 

			  
import numpy as np
def generate_ma_time_series(response_function, n_obs):
    """
    สร้างกราฟแสงจากโมเดล MA ด้วยฟังก์ชันการตอบสนอง (response function).

    Parameters:
    - response_function: ฟังก์ชันการตอบสนอง (response function) ของโมเดล MA.
    - n_obs: จำนวนจุดข้อมูลของกราฟแสงที่ต้องการสร้าง

    ผลลัพธ์:
    - time_series: time series ของกราฟแสง
    """

    # สร้าง white noise
    white_noise = np.random.normal(size=n_obs)

    # คำนวนฟังก์ชันการตอบสนอง
    response_values = response_function(np.arange(n_obs))

    # จัดการ array 'white noise'  สำหรับการ  convolve  เพื่อให้ กราฟแสงที่ได้ออกมามีความยาวเท่ากับ n_obs ด้วยการใส่ "0" ใน array ก่อนการ  convolve
    white_noise_padded = np.concatenate((white_noise, np.zeros(n_obs - 1)))

    #สร้างกราฟแสงด้วยการ convolve 'white noise' กับ ฟังก์ชันการตอบสนอง ตามสมการโมเดล MA
    time_series = np.convolve(white_noise_padded, response_values, mode='valid')

    return time_series[:n_obs]  # ตัดทอน array 'time_series' ตามความยาว 'n_obs'

จากฟังก์ชันด้านบนเราจะแสดงตัวอย่างการใช้งานโดยฟังก์ชันการตอบสนองเป็นสมการ exponential \(t^2\) และให้จำวนวนจุดข้อมูลเท่ากับ 1000 โดยตัวอย่างโค้ดและผลลัพธ์จะเป็นดังนี้ 

			  
# ตัวอย่างการใช้งานฟังก์ชันเพื่อสร้างกราฟแสงด้วยโมเดล MA

# กำหนดตัวอย่างฟังก์ชันการตอบสนองอย่างง่าย (เช่น สมการ exponential)
def response_function(t):
    return (t^1)

n_obs = 1000  # กำหนดจำนวนจุดข้อมูลของกราฟแสง
# ใช้ฟังก์ชันในการสร้างกราฟแสง

light_curve = generate_ma_time_series(response_function, n_obs)

# สร้างกราฟของกราฟแสง
plt.plot(light_curve)
plt.title('Time Series Generated with Moving Average (MA) Model')
plt.xlabel('Time')
plt.ylabel('Amplitude')
plt.show()
