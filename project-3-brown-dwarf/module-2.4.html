ข้อมูลการสังเกตการณ์ดาวแคระน้ำตาล

การจัดการกับข้อมูลที่หายไป (Missing value)

จากข้อมูลจะเห็นได้ว่าค่า Magnitue ของแต่ล่ะ Filter ของดาวบางส่วนขาดหายไป (ค่า NaN) ดังรูปที่ 1 และหากพิจารณา Data fame 'all_df' ก่อนหน้านี้ จะพบว่าค่าดัชนีสีมีค่าเป็น NaN อยู่เช่นกัน ดังรูปที่ 2

ในการจัดการกับค่าที่หายไป เราจะใช้ Iterative Imputer จาก Library 'Scikit-learn' ในการเติมค่าที่หายไป วิธีการนี้อาศัยการทำนายค่าด้วยการวิเคราะห์แบบถดถอย (Regression) ด้วย Machine learning โดยวิเคราะห์จาก function หลายมิติที่ประกอบด้วยความสัมพันธ์ระหว่างข้อมูลของค่าในหลัก (column) อื่นๆประกอบการทำนาย

ขั้นแรกของการทำ Iterative Imputer จะเริ่มจากการสร้าง Data frame 2 ตัว คือ df_imp_tr และ df_imp_eva สำหรับนำไปฝึกฝน (train) และประเมินผล (evaluate) แบบจำลอง Iterative Imputer โดยอิงจากข้อมูล all_df เดิม และตัดข้อมูลแถว (row) ที่เป็น 'NaN' ออก 

			
#ทำการกำจัด ข้อมูลในแถว(row)ที่มี "NaN" ออกไปทั้งหมด ด้วยคำสั่ง '.dropna()'
df_imp_tr = all_df.dropna()
df_imp_eva = all_df.dropna()
#ทำการตัด column ของ label class ออก เนื่องจากไม่มีความจำเป็นขั้นตอนนี้
df_imp_tr = df_imp_tr.drop('label', axis = 1)
df_imp_eva = df_imp_eva.drop('label', axis = 1)
display(df_imp_eva)
display(df_imp_tr)

เมื่อให้แสดงผลข้อมูล ทั้ง df_imp_tr และ df_imp_eva ยังคงมีข้อมูลที่เหมือนกันทุกประการ ดังนี้

จากนั้นจะทำการสุ่มข้อมูล 5 % ใน df_imp_tr ให้เป็น 'NaN' value เพื่อใช้ในการฝึกฝนและสร้างแบบจำลอง Imputer ของเรา

หลายคนอาจสงสัยว่าทำไมเราจึงต้องตัด 'NaN' value ใน all_df ก่อน แล้วจึงค่อยสุ่ม 5% ให้เป็น NaN อีกรอบ ก่อนนำข้อมูลที่ได้ไปใช้ฝึกฝนสร้างแบบจำลอง ทำไมจึงไม่ใช้ all_df ที่มี 'NaN' value อยู่แล้วในการฝึกฝน เหตุผลก็คือเราไม่รู้ค่าแท้จริงของ 'NaN' ใน all_df เราจึงไม่อาจรู้หรือประเมินผลได้เลยว่าแบบจำลอง Iterative Imputer ที่ได้ออกมานั้นมีประสิทธิภาพแค่ไหน แต่ถ้าใช้ df_imp_tr ที่ถูกสุ่มให้ข้อมูลบางตัวเป็น 'NaN' เราสามารถประเมินความแม่นยำของแบบจำลองในการ Imputation (หรือการทำนายค่า 'NaN' เป็นค่าจริง) ได้ โดยนำค่าที่ทำนายได้ไปเปรียบเทียบกับค่าจริงใน df_imp_eva (ซึ่งมีข้อมูลที่เหมือนกันทุกประการกับ df_imp_tr เดิมก่อนที่จะถูกสุ่มข้อมูล 5 % เป็นค่า 'NaN')

ตัวอย่างโค้ดของกระบวนการดังกล่าวเป็นดังนี้ 

			
# สร้าง array ของ True, False ที่มี shape ตรงกับ df_imp_tr ทุกประการ โดยสุ่มให้มีค่า True 5%(0.05) และค่า False 95%(0.95) 
missing_mask = np.random.choice([True, False], size=df_imp_tr.shape, p=[0.05, 0.95])
# ให้ค่าในตำแหน่งของข้อมูลใน df_imp_tr ซึ่งตรงกับ missing_mask ที่มีค่าเป็น True เท่ากับ 'NaN' value ซึ่งจะทำให้ค่าใน df_imp_tr มีค่าเท่ากับ 'NaN' value  5% พอดี
df_imp_tr[missing_mask] = np.nan
display(df_imp_tr)

ผลลัพธ์ที่ได้คือ

ต่อไปเป็นขั้นตอนการฝึกฝนและสร้างแบบจำลอง Iterative Imputer โดยเราจะใช้ Extra Trees Regressor เป็น estimator หรือตัวทำนายผลของเรา และกำหนดค่าตัวแปร (parameter) ที่สำคัญใน Extra Trees Regressor เป็น n_estimators = 150, max_features = 14, max_depth=15, min_samples_split =12, initial_strategy = 'median' และให้มีการวนซ้ำใน Iterative Imputer มากที่สุด 20 ครั้ง (max_iter = 20) ในการใช้งานจริงตัวแปรเหล่านี้ควรจะต้องมีการ fine-tune เพื่อให้ได้แบบจำลองที่ดีที่สุด แต่เนื่องจากเราจะทำแบบฝึกหัดต่อไปตาม Avdeeva 2023 เราจะเลือกใช้ตัวแปรต่างๆ ตามบทความนี้ไปเลย (ตามค่าด้านบน) ความหมายของ parameter ต่างๆ สามารถอ่านเพิ่มเติมได้ใน Extra Trees Regressor และตัวอย่างการ fine-ture ตัวแปรสามารถอ่านเพิ่มเติมได้ใน Iterative Imputation

ในขั้นตอนการประเมิณผลแบบจำลอง Iterative Imputer เราจะใช้คำสั้ง '.isclose' ใน Library numpy โดยเป็นการเปรียบเทียบค่าที่ถูก Imputation ใน df_imp_tr กับค่าแท้จริงใน df_imp_eva ว่าใกล้เคียงกันมากน้อยแค่ไหน จากนั้นใช้คำสั้ง '.mean' คำนวนความแม่นยำเฉลี่ย

โดยตัวอย่างโค้ดจะเป็นดังนี้ 

			
from sklearn.experimental import enable_iterative_imputer
from sklearn.impute import IterativeImputer
from sklearn.ensemble import ExtraTreesRegressor

#ตั้งค่า Parameter ต่างๆใน IterativeImputer
imp = IterativeImputer(estimator=ExtraTreesRegressor(n_estimators =150, max_features = 14, max_depth=15,
                                                                           min_samples_split =12), initial_strategy = 'median', max_iter =20)
#การฝึกฝน IterativeImpute ที่เราสร้าางขึ้นด้วยข้อมูล "df_imp_tr" ด้วยคำสั่ง ".fit"
imp.fit(df_imp_tr)

# ทำการ Imputation ชุดข้อมูลของเราด้วย imputing machince learning ทีถูกฝึกฝนมาแล้ว ด้วยคำสั่ง '.transform'
Im_data = pd.DataFrame(imp.transform(df_imp_tr))


# คำนวนความแม่นยำของIterativeImputerของเรา
accuracy = np.mean(np.isclose(df_imp_tr[missing_mask], Im_data[missing_mask], equal_nan=True))
print("ค่าความแม่ยำของการ Imputation :", accuracy*100)
display(Im_data)

ผลลัพธ์ที่ได้เป็นดังนี้

จะเห็นว่าค่าความแม่นยำในการ Imputation (เปลี่ยนค่า 'NaN' เป็นตัวเลขจริง) อยูที่ ~78% และค่า 'NaN' ใน Data frame นั้นถูกทำให้เป็นค่าตัวเลขจริงทั้งหมดแล้ว โดยเราเรียก Data frame ใหม่หลังทำ Imputation ซึ่งข้อมูลไม่มีค่า 'NaN' แล้วว่า Im_data
