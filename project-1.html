ดาวแคระน้ำตาล(Brown Dwarfs)คือ?

ดาวแคระน้ำตาลคือดาวที่มีมวลมากกว่าดาวเคราะห์ แต่น้อยกว่าดาวฤกษ์ ในทางดาราศาสตร์ ดาวแคระน้ำตาลถูกจัดว่าเป็นวัตถุมวลต่ำกว่าดาวฤกษ์ (substellar object) ซึ่งถูกคาดการณ์การมีอยู่ทางทฤษฎีครั้งแรกโดย Dr. Shiv S. Kumar ในช่วงปี 1960s ในช่วงแรก เราเรียกดาวแคระห์น้ำตาลว่า “ดาวแคระดำ” อันเนื่องมาจากขนาดที่เล็กและอุณหภูมิที่ต่ำ ทำให้ยากต่อการค้นพบจากการสังเกตการณ์ จนกระทั้งในปี 1994-1995 มีการค้นพบดาวแคระน้ำตาล 2 ดวงคือ Gliese 229b(ถูกค้นพบก่อน?) and Teide 1(ตีพิมพ์ก่อน?)จากการสังเกตการณ์ในย่านคลื่นอินฟราเรด

ทำไมดาวแคระน้ำตาลจึงถูกค้นพบได้ยาก?

เนื่องจากมวลของดาวแคระน้ำตาลนั้นมีค่ามากกว่ามวลของดาวเคราะห์ แต่น้อยกว่าดาวฤกษ์ (มีมวลระหว่าง 13 ถึง 80 เท่าของดาวพฤหัสบดี) ซึ่งอยู่ในช่วงที่ทำให้เกิดปฎิกริยาฟิวชันของดิวเทอเรียม (Deuterium fusion) ที่แกนกลางของดาว แต่ไม่มากพอที่จะทำให้เกิดเกิดปฎิกริยาฟิวชันของไฮโดรเจนแบบดาวฤกษ์ (มีมวล > 80 เท่าของดาวพฤหัส) ทำให้บางครั้งนักดาราศาสร์เรียกมันว่า “ดาวฤกษ์ที่ล้มเหลว (failed stars)” และด้วยเหตุนี้เองทำให้พวกมันมีค่ากำลังส่องสว่าง (luminosity) และอุณหภมูิที่ต่ำ อีกทั้งยังมีขนาดเล็ก ทำให้ถูกค้นพบในย่านคลื่นอินฟราเรดโดยทั่วไป

การจำแนกดาวแคระแคระน้ำตาล

ปกตินักดาราศาสตร์จะใช้การวิเคราะห์เส้นสเปกตรัม (Spectrograph) ในการจัดหมวดหมู่ของดาวตั้งแต่ดาวฤกษ์ไปจนถึงดาวแคระตามประเภทต่าง ๆ การวิเคราะห์เส้นสเปกตรัมสามารถทำได้โดยใช้เทคนิค Spectroscopy ซึ่งเป็นการแยกแสงที่ได้มาจากการสังเกตการณ์ด้วยการใช้เกรตติ้งเลี้ยวเบน (Diffraction grating) เพื่อให้เห็นค่าความเข้มแสงของแต่ละย่านคลื่น รูปร่างของเส้นสเปกตรัมนั้นจะเปลี่ยนไปตามคุณสมบัติของดาว เช่น อุณหภูมิพื้นผิวของดาวและองค์ประกอบธาตุในดาว ค่าอุณหภูมิพื้นผิวของดาวจะขึ้นอยู่กับยอดคลื่นของเส้นสเปกตรัมที่มีรูปร่างตามกฎการแผ่รังสีของวัตถุดำ (Blackbody radiation) ส่วนค่าองค์ประกอบธาตุของดาวจะมีผลต่อการปรากฏของเส้นการดูดกลืน (Absorbtion line) ที่ความยาวคลื่นต่าง ๆบนเส้นสเปกตรัม ทำให้เราสามารถใช้การวิเคราะห์เส้นสเปกตรัมในการวัดค่าอุณหภูมิพื้นผิวของและความเป็นโลหะของดวงดาวได้ ซึ่งเราใช้ข้อมูลเหล่านี้ในการจำแนกชนิดดาวออกตาม spectrum class ในแต่ละหมวดหมู่ตามคุณสมบัติของดาว Spectrum class มาตราฐานของดาวมีอยู่ 7 ประเภทหลัก ๆ ได้แก่ O, B, A, F, G, K และ M เรียงตามอุณหภูมิพื้นผิวจากร้อนสุดไปหาเย็นที่สุดดังรูปที่ 1. แต่เมื่อพิจารณาการมีอยู่ของดาวแคระน้ำตาลซึ่งมีอุณหภูมิต่ำกว่าประเภท M และมีค่าความเป็นโลหะที่มากกว่าดาวกฤษ์ทั่วไป จึงมีการแบ่งประเภทของ Spectrum class ย่อยเพิ่มขึ้นมาอีก 3 ประเภทคือ L, T และ Y ดังที่เห็นในรูปที่ 2. ซึ่งในบางครั้งนักดาราศาสตร์ยังจัดว่าดาวแคระแดง (Red dwarf) ที่อยู่ในประเภท M ช่วงปลาย (M6-M9) เป็นดาวแคระแคระน้ำตาลด้วย ขึ้นอยู่กับ Absorbtion line ของดาวแคระแดง ว่ามีเส้นสเปกตรัมของธาตุ Li ที่เกิดมาจากปฎิกริยาฟิวชันของดิวเทอเรียมหรือไม่

นอกจากการใช้เทคนิค Spectroscopy เพื่อจำแนกประเภทดาวแคระน้ำตาลแล้ว เราก็ยังสามารถใช้วิธีทาง Photometric ในการจำแนกประเภทดาวแคระน้ำตาลได้ด้วย โดยเทคนิคนี้ถูกเสนอขึ้นโดย Nathalie Skrzypek ซึ่งเริ่มต้นในปี 2013 และได้ตีพิมพ์ในปี 2015 (Skrzypek et al. 2015) ซึ่งจำแนกดาวแคระน้ำตาลโดยการ fit graph ตาม template ของ color index ดังที่แสดงในรูปที่ 3.

การหาค่า color index ทำได้โดยการคำนวณความแตกต่างของ photon flux ระหว่าง 2 ย่านคลื่นที่เราสนใจ โดยค่า photon flux สามารถหาได้จากการใช้ filter กรองแสงในคลื่นย่านอื่นออกไป ซึ่งย่านคลื่นที่ครอบคลุมในแต่ละ filter เป็นดังตัวอย่างในรูปที่ 4. จะเห็นได้ว่าเมื่อใช้ ‘V filter (เส้นสีเขียว)’ ค่า photon flux ที่วัดได้จะอยู่ในช่วงความยาวคลื่น 450 nm to 650 nm ทำให้เราจะได้ค่า Magnitude ของดาวดวงนั้นในย่านคลื่นนี้ การคำนวณค่า color index ทำได้โดยการนำ Magnitude ของ 2 filter มาลบกัน ในตัวอย่างรูปที่ 3. แกน y คือค่า color index ระหว่าง ‘i filter’ และ ‘Z filter’ เมื่อได้ค่า color index แล้ว (Skrzypek et al. 2015) ได้ใช้วิธีการทางสถิติในการสร้าง template สำหรับ fit graph ระหว่างชนิดของดาวแคระน้ำตาลกับค่า color index โดยรายละเอียดสามารถอ่านเพิ่มเติมได้ในเปเปอร์ https://www.aanda.org/articles/aa/pdf/2015/02/aa24570-14.pdf
