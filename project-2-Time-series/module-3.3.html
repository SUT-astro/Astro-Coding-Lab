โมเดล PSD และ กราฟ fiting

ในบทนี้เราจะแสดงตัวกราฟการ fitting กราฟ PSD ด้วย สมการกำลัง (power-law equation) ซึงเป็นสมการที่ถูกใช้อย่างกว้างขวางในการอธิบายฟิสิกส์(ความแปรปรวน)ของ AGN ในเบื้องต้น โดยจะสามารถเขียนเป็นสมการได้ดังนี้

$$ \text{PSD} = \frac{k}{f^{\beta}} $$
  • PSD คือ the Power Spectral Density
  • \( f \) คือ ความถี่
  • \( k \) คือ ค่าคงที่การทำให้เป็นมาตรฐาน (normalization constant)
  • \( \beta \) คือ ดัชนี power-law

ในสมการนี้ PSD จะแสดงถึงปริมาณกำลังความแปรปรวนที่มีอยู่ภายในแต่ช่วงความถี่ ดัชนี power-law (\( \beta \)) แสดงลักษณะความชันของ PSD และกำหนดว่าพลังงานความแปรปรวนจะเปลี่ยนแปลงตามความถี่อย่างไร

ถัดไปจะเป็นการแสดงตัวอย่างการ fitting กราฟ PSD จากข้อมูลสังเกตการณ์จริง

ขั้นตอนแรกจะเป็นการเรียกใช้ข้อมูลด้วย Pandas library ดังที่เคยทำมาแล้วในแบบผฝึกหัดที่ 1 โดยตัวอย่างโค้ดจะเป็นดังนี้

***จะเห็นว่าด้านบนกราฟจะมีการเตือนอยู่ไม่จำเป็นต้องสนใจ มันเตือนว่าเจากำลังเขียนทับค่าใน Data frame ของเราอยู่

ขั้นตอนถัดไปจะเป็นการ re-bin ข้อมูล (กำหนดช่วงความกว้างของแต่จุดใหม่) ข้อมูลของเรานั้นมี time step อยู่ที่ 1 วินาที่ เราจะ re-bin ให้ time step อยู่ 10 วินาที โดยใช้คำสัง binned_statistic ใน library Scipy ซึ่งการ rebin นั้นสามารถช่วยลดสัญญาณรบกวนในข้อมูลได้ โดยตัวอย่างโค้ดจะเป็นดังนี้ 

          
from scipy.stats import binned_statistic

# นำ column data['time'] และ data['time'] ใน Data frame แปลงเป็น array เนื่องจากคำสั้ง binned_statistic นั้น input ต้องเป็น array 
ttt = data['time'].to_numpy()
Count_rate = data['time']to_numpy()

new_lc_bin, new_time_bin, _ = binned_statistic(ttt, Count_rate, statistic='mean', bins = np.arange(ttt[0],ttt[-1], 10))

new_time_bin = new_time_bin[:-1]

ถัดไปจะแสดงตัวอย่างการ fitting PSD ด้วยสมการกำลัง (power-law equation) โดยใช้คำสั่ง 'curve_fit' ใน library Scipy

ตัวอย่างขั้นตอนการเขียนโค้ดเพื่อ fitting PSD จะเป็นดังนี้ 

          
from scipy.optimize import curve_fit

# กำหนดสมการเพื่อใช้ในการ fitting กราฟ
def power_law_function(x, k, b): 
    """
    กำหนดให้สมการเป็นสมการกำลัง (power-law equation)  
    - x  คือ ความถี่
    - k  คือ ค่าคงที่การทำให้เป็นมาตรฐาน
    - b คือ  ดัชนี power-law
    """
    return ( k/(x**(b)) )

# เลือกตัวแปร x แลพเพื่อใช้การ fitting กราฟ
x, y = f, PSD

# ใช้งานคำสั้ง curve_fit ด้วยสมการกำลังเพื่อ fitting กราฟ
popt, _ = curve_fit( power_law_function, x, y )

# สรุปค่าพารามิเตอร์ ที่ได้จาการ fiting กราฟ
k, b= popt
#แสดงค่าพารามิเตอร์ 
print(" ค่าคงที่การทำให้เป็นมาตรฐาน เท่ากับ % s " %(k))
print(" ดัชนี power-law เท่ากับ % s " %(b))


# พล็อตกราฟ PSD ดั้งเดิม กับ PSD ที่ได้จากการ fiting กราฟ

#พล็อตกราฟ PSD ดั้งเดิม
plt.plot(f, PSD)
#สร้าง PSD จากพารามิเตอร์ที่ได้จากการ fitting กราฟ 
y_line = power_law_function(x, k, b)
#PSD ที่ได้จากการ fiting กราฟ
plt.plot(x, y_line, '--', color='red', label= r'P ∝ $f^{%s}$' %b)
plt.legend(fontsize= 15)
plt.xscale('log')
plt.yscale('log')

จากผลลัพธ์ข้างต้นจะเห็นว่าเมื่อเปรียบ PSD ดั้งเดิมกับ PSD ที่สร้างจากพารามิเตอร์ที่ได้จากการ fitting กราฟนั้น ณ ที่ความถี่ต่ำจะเห็นว่า PSD ดั้งเดิม นั้นมีค่ามากว่า PSD ที่ได้จากการ fitting เป็นผลมาจาก White noise ซึ่งจะทำให้ PSD มีค่ามากกว่าปรกติ ซึ่งคล้ายกับกรณีที่เรานำ White nosie รวมกับ Red noise ใน section ที่แล้ว

ดังนี้เราจะ fitting กราฟให้ โดยเปลี่ยนสมการที่ใช้ในการ fitting โดยเพิ่มพจน์ของ White noise ('w') ซึ่งเป็นค่าคงที่ลงไปในสมการกำลัง (power-law equation) ของเรา ตัวอย่างขั้นตอนการเขียนโค้ดจะเป็นดังนี้ 

          
from scipy.optimize import curve_fit

# กำหนดสมการเพื่อใช้ในการ fitting กราฟ
def power_law_with_white_noise_function(x, k, b, w): 
    """
    กำหนดให้สมการเป็นสมการกำลัง (power-law equation)  ทีมี White noise รวมอยู่ด้วย
    - x  คือ ความถี่
    - k  คือ ค่าคงที่การทำให้เป็นมาตรฐาน
    - b คือ  ดัชนี power-law
    - w คือ  พจน์ของ white noise
    """
    return ( k/(x**(b)) ) + w

# เลือกตัวแปร x แลพเพื่อใช้การ fitting กราฟ
x, y = f, PSD

# ใช้งานคำสั้ง curve_fit ด้วยสมการกำลังเพื่อ fitting กราฟ
popt, _ = curve_fit(  power_law_with_white_noise_function, x, y )

# สรุปค่าพารามิเตอร์ ที่ได้จาการ fiting กราฟ
k, b, w= popt
#แสดงค่าพารามิเตอร์ 
print(" ค่าคงที่การทำให้เป็นมาตรฐาน เท่ากับ % s " %(k))
print(" ดัชนี power-law เท่ากับ % s " %(b))
print(" White noise เป็นเป็นค่าคงที่ เท่ากับ % s " %(w))



# พล็อตกราฟ PSD ดั้งเดิม กับ PSD ที่ได้จากการ fiting กราฟ

#พล็อตกราฟ PSD ดั้งเดิม
plt.plot(f, PSD)
#สร้าง PSD จากพารามิเตอร์ที่ได้จากการ fitting กราฟ 
y_line = power_law_with_white_noise_function(x, k, b, w)
#พล็อตPSD ที่ได้จากการ fiting กราฟ
plt.plot(x, y_line, '--', color='red', label= r'PSD with white noise')
#สร้างPSD ตามสมการกำลัง
y_line1 = power_law_function(x, k, b)
plt.plot(x, y_line1, '-.', color='blue', label= r'Original PSD [P ∝ $f^{%s}$]' %b)
