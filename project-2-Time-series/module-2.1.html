Autoregressive Model (AR model)

Autoregressive Model เป็นโมเดลทางสถิติที่ใช้ในการวิเคราะห์และทำนายข้อมูลชุดตัวเลขในโดเมนเวลา โมเดลนี้ได้รับการใช้งานอย่างแพร่หลายในการศึกษาและการทำนายแนวโน้มของชุดข้อมูลที่มีการเปลี่ยนแปลงตามเวลา เช่น ราคาหุ้น, อุณหภูมิประจำวัน, หรือปริมาณการขายของสินค้า อีกทั้งมันยังสามารถใช้เป็นเครื่องมือในการสร้าง light curves ( Fourier Methods by Matteo Bachetti & Daniela Huppenkothen) วัตถุทางดาราศาสตร์บางประเภท โดยเฉพาะวัตถุที่มี light curves เป็น Stationary light curves

โดย Autoregressive Model ถูกสร้างขึ้นจากแนวคิดที่ว่าข้อมูลในปัจจุบันนั้นสามารถอธิบายได้ด้วยชุดข้อมูลในอดีตและสามารถนำมาทำนายสิ่งที่จะเกิดขึ้นในอนาคตได้อีกด้วย โดยสามารถเขียนเป็นสมการได้ดังนี้

$$ x_t = \sum_{i=1}^{p} \phi_i x_{t-i} + \varepsilon_t $$

เมื่อ :

  • \(x_t\): เส้นโค้งแสง ณ เวลา t ใดๆ
  • \(\phi_i\): ค่าสัมประสิทธิ์ของโมเดล AR \(\phi_1\) หมายถึงค่าน้ำหนักของค่าที่ในอดีดผ่านมาล่าสุด และ \(\phi_2\) หมายถึงน้ำหนักของค่าอดีตในอันดับสองของค่าปัจจุบัน
  • \(p\): ค่า Order ของโมเดล AR ซึ่งจะกำหนดจำนวนของค่าในอดีตที่ใช้ในการทำนายค่าปัจจุบัน แต่กรณีของเราหมายถึงจำนวนของค่าในอดีตที่ต้องการใช้สร้าง light curves ในปัจจุบัน
  • \(\varepsilon_t\): ค่าความคลาดเคลื่อนแบบสุ่ม(white noise)

ตัวอย่างของฟังก์ชันโค้ดโมเดล AR ที่ใช้ในการการสร้างกราฟแสงจะเป็นดังนี้ 

			  
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

def generate_AR_series(n_obs, phi_values):
    """
    สร้างกราฟแสงจากโมเดล AR(p)

    Parameters:
    - n_obs: จำนวนจุดของกราฟแสงที่ต้องการสร้าง
    - phi_values:  ค่าสัมประสิทธิ์ของโมเดล AR(p) ใดๆ (list หรือ array)

    ผลลัพธ์:
    - time_series: time series ของกราฟแสง
    """

    p = len(phi_values)    # Order ของโมเดล AR อยู่กับ array  'phi_values' โดยค่าแรกของarray คือ phi_1 และถัดมาจะเป็น phi_2, phi_3, ...
    np.random.seed(42)

    #สร้าง wihte noise
    white_noise = np.random.normal(size=n_obs)

    # สร้าง time series โดยขึ้นอยู่กับโมเดล AR(p) ใดๆ 
    time_series = [white_noise[i] for i in range(p)]

    #สร้างกราฟแสงโดยใช้สมการของโมเดล AR
    for t in range(p, n_obs):
        X_t = sum(phi * time_series[t-i-1] for i, phi in enumerate(phi_values)) + white_noise[t]
        time_series.append(X_t)

    return time_series

จากฟังก์ชันด้านบนเราจะแสดงตัวอย่างการใช้งานโดย จะสร้างกราฟแสงด้วยโมเดล AR(1)และ AR(2) โดยเราจะให้จำนวนจุดของกราฟแสงเท่ากับ 1024 ค่าสัมประสิทธิ์ของโมเดล AR(1) นั้นมีค่าเท่ากับ -0.2 และค่าสัมประสิทธิ์ของโมเดล AR(2) นั้นมีค่าเท่ากับ 0.4 และ 0.5 ตามลำดับ โดยตัวอย่างโค้ดและผลลัพธ์จะเป็นดังนี้

โมเดล AR(1) 

        
# ตัวอย่างการใช้งานฟังก์ชันเพื่อสร้างกราฟแสงด้วยโมเดล AR(1)
n_obs_example = 1024 # กำหนดจำนวนจุดข้อมูลของกราฟแสง
phi_values_example = [-0.2] # กำหนดค่าสัมประสิทธิ์ของโมเดล 

# ใช้ฟังก์ชันในการสร้างกราฟแสง

light_curve = generate_AR_series(n_obs_example, phi_values_example)

# พล็อตกราฟของกราฟแสง
plt.plot(light_curve, label='Generated Time Series (AR(1))')
plt.legend()
plt.title('Generated Time Series from AR(1) Model')
plt.show()
