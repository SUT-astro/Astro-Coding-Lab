กราฟแสง (Light Curve)

Light curve คือ ?

Light curve หรือ กราฟแสง เป็นการแสดงผลการวัดความเข้มของแสงในโดเมนของเวลา (แกน X เป็นเวลา และแกน Y เป็นการรับแสงหรือความสว่างที่ถูกตรวจวัดได้) โดยเราอาจแบ่งประเภทของการตรวจวัด light curve ได้ดังนี้

  • Light curve เป็นเครื่องมือสำคัญที่ใช้ในการค้นพบวัตถุใหม่ ๆ ในทางดารารศาสตร์ ซึ่งการสำรวจLight curve ของวัตถุที่มีความแปรปวรน้อยบ่อยครั้งช่วยให้นักดาราศาสตร์ค้นพบวัตถุที่น่าสนใจและเข้าใจเกี่ยวกับลักษณะของพวกเขา
  • การวิเคราะห์Light curveสามารถใช้ในการระบุคุณลักษณะต่างๆของวัตถุทางดาราศาสตร์ เช่น การตรวจจับรูปแบบการเคลื่อนที่ของดาวเคราะห์, การทำนายช่วงการเกิดเหตุการณ์สำคัญ(e.g. supernova)
  • นักดาราศาสตร์ยังสามารถใช้ Light curve เพื่อศึกษาเชิงทฤษฎีและการทดลอง ในบางครั้งมีการนำข้อมูลแสงมาวิเคราะห์เพื่อสามารถยืนยันหรือปฏิเสธทฤษฎีทางฟิสิกส์ได้อีกด้วย
  • การวิเคราะห์ฟกราฟแสงนั้นสามารถใช้ระบุลักษณะของวัตถุที่ไม่สามารถมองเห็นได้ด้วยตาเปล่า เช่น หลุมดำ, วัตถุทีแผ่รังสีในย่านคลื่น X-ray, หรือวัตถุที่มีการรับแสงได้น้อยและไม่อยู่ใช่ช่วงแสงที่สามารถมองเห็นได้ด้วยตาเปล่า

ค่าเฉลี่ย (Mean) ความแปรปรวน (Variance) และ Autocovariance

ค่าเฉลี่ย (Mean): ค่าเฉลี่ยของ Light Curve คือค่าเฉลี่ยของการรับแสงหรือความสว่างที่ถูกวัดได้ในช่วงเวลาที่กำหนด การคำนวณค่าเฉลี่ยสามารถช่วยให้เราเข้าใจว่ามีความสว่างเฉลี่ยเท่าใดในระหว่างช่วงเวลาที่สังเกตุ

ความแปรปรวน (Variance): ความแปรปรวนของ Light Curve บ่งบอกถึงกระจายตัวของข้อมูล ค่าความแปรปรวนมีความสัมพันธ์กับค่าความคลาดเคลื่อนของข้อมูลจากค่าเฉลี่ย ถ้าความแปรปรวนมีค่ามาก แสดงว่ามีความคลาดเคลื่อนมากในข้อมูล

Autocovariance : Autocovariance คือการวัดความสัมพันธ์ของค่า ณ เวลาใดๆ Light Curve กับค่าในเวลาก่อนหน้านั้น ค่า Autocovariance มีประโยชน์ในการหารูปแบบซ้ำ ๆ ที่เกิดขึ้นในข้อมูล

การใช้ค่าสถิติเหล่านี้ในการวิเคราะห์ Light Curve ช่วยให้เราทำความเข้าใจกับพฤติกรรมแสงของดาวฤกษ์ได้ลึกซึ้งมากขึ้น ซึ่งมีประโยชน์ในการศึกษาแนวโน้ม, การจับความสัมพันธ์กับตัวแปรอื่น ๆ, หรือการตรวจจับรูปแบบที่ซับซ้อนในข้อมูล Light Curve ที่มีความสำคัญสำหรับการวิเคราะห์ดาวฤกษ์และระบบดาวฤกษ์ต่าง ๆ ในจักรวาลได้ของเรา

Stationary and Non-stationary light curves

Stationary light curves

Stationary light curves คือ light curves ของความสว่างของวัตถุทางดาราศาสตร์ในช่วงเวลาหนึ่ง โดยที่คุณสมบัติทางสถิติโดยรวมคือค่าความสว่างของมันไม่เปลี่ยนแปลงตามเวลา หรือก็คือ รูปแบบความสว่างจะเกิดซ้ำในลักษณะที่คาดเดาได้ ซึ่งหมายความว่าความสว่างโดยเฉลี่ย ค่าความแปรปรวน (ความสว่างเปลี่ยนแปลงไปมากน้อยเพียงใด) และรูปการเปลี่ยนแปลงใดๆ ตามเวลาจะมีค่าคงที่ทั้งหมด

คุณลักษณะของ Stationary light curves

  • ไม่มีการเปลี่ยนแปลงแนวโน้ม : ความสว่างโดยรวมจะมีแนวโน้มแนว(trend)ที่ไม่เปลี่ยนแปลง ในระยะยาวใดๆ ไม่ว่าจะเพิ่มขึ้นหรือลดลง
  • ความแปรปรวน (Variance) คงที่: ค่าความผันผวนของความสว่างจะยังคงสม่ำเสมอเมื่อเวลาผ่านไป
  • รูปแบบที่เกิดซ้ำ: สามารถคาดเดารูปแบบความสว่างที่แปรผันตามเวลาได้ แม้ว่าเมื่อเวลาผ่านไปแต่ละรูปแบบอาจไม่เหมือนกันทุกประการ

อีกทั้ง Stationary light curves สามารถใช้เป็นเครื่องมือสำหรับนักดาราศาสตร์ที่ต้องการศึกษาคุณลักษณะต่างๆของวัตถุท้องฟ้าได้ดังนี้

ระบุประเภทของดาวต่างๆ: ดาวประเภทต่างๆ นั้น มีรูปแบบลักษณะความแปรปรวนเฉพาะที่แตกต่างกัน ซึ่งสามารถนำไปใช้ในการจำแนกประเภทดาวฤกษ์เหล่านั้นได้

การศึกษากระบวนการทางกายภาพในดาวฤกษ์: ความแปรผันของความสว่างของดาวฤกษ์อาจเกิดจากกระบวนการทางกายภาพต่างๆ เช่น การเต้นของดาวฤกษ์(การยุบและขยายที่ต้องการักษาสมดุลของตัวมันเอง [Stellar pulsation]) การหมุน และสนามแม่เหล็กของดาว ด้วยการศึกษากราฟแสงของพวกมัน นักดาราศาสตร์สามารถศึกษาดพิ่มเติมเกี่ยวกับกระบวนการเหล่านี้ได้

การตรวจจับดาวเคราะห์นอกระบบ: ดาวเคราะห์นอกระบบสุริยะบางดวงอาจทำให้ความสว่างของดาวฤกษ์แม่ลดลงเป็นระยะๆ ขณะที่มันเคลื่อนผ่านหน้าดาวฤกษ์แม่ ด้วยการค้นหาการลดลงเหล่านี้ในกราฟแสงของดาวฤกษ์ นักดาราศาสตร์สามารถตรวจจับการมีอยู่ของดาวเคราะห์นอกระบบได้ และอาจจะสามารถระบุธาตุที่เป็นองประกอบของดาวฤกษ์นั้นๆได้อีกด้วย

Non-stationary light curves

Non-stationary light curves เป็น light curve ของวัตถุทางดาราศาสตร์ในช่วงเวลาใดเวลาหนึ่งหนึ่ง โดยที่คุณสมบัติทางสถิติโดยรวมจะมีการแปรผันของความสว่างเปลี่ยนแปลงตามเวลา หรือก็คือ รูปแบบความสว่างนั้นจะไม่เกิดซ้ำในลักษณะที่สามารถคาดเดาได้ ซึ่งหมายความว่าความสว่างโดยเฉลี่ย ค่าความแปรปรวน และรูปการเปลี่ยนแปลงใดๆ ล้วนสามารถเปลี่ยนแปลงได้ตลอดเวลา

คุณลักษณะของ Non-Stationary light curves

  • การเปลี่ยนแปลงแนวโน้ม: ความสว่างโดยรวมสามารถแสดงแนวโน้มว่าสามารถเปลี่ยนแปลงได้ ไม่ว่าจะมีค่าเพิ่มขึ้นหรือลดลง
  • ความแปรปรวนแปรผันตามเวลา: จำนวนความผันผวนของความสว่างสามารถเปลี่ยนแปลงได้ตลอดเวลา
  • ไม่มีรูปแบบที่เกิดซ้ำ: ความแปรผันของความสว่างจะไม่เกิดซ้ำในลักษณะที่คาดเดาได้ แม้ว่าอาจมีโครงสร้างดังเดิมอยู่บ้างก็

และNon-Stationary light curves สามารถใช้เป็นเครื่องมือสำหรับนักดาราศาสตร์ที่ต้องการศึกษาคุณลักษณะและเหตุการณ์ต่างๆของวัตถุท้องฟ้าได้ดังนี้

การศึกษาวัตถุที่มีพฤติกรรมซับซ้อน: มีวัตถุทางดาราศาสตร์จำนวนมาก เช่น ควาซาร์ บลาซาร์ และหลุมดำที่กำลังสะสมอยู่ มีเส้นโค้งแสงที่ไม่คงที่ ด้วยการศึกษาเส้นโค้งแสงเหล่านี้ เราสามารถเรียนรู้เพิ่มเติมเกี่ยวกับกระบวนการทางกายภาพที่ซับซ้อนที่ภายในวัตถุเหล่านี้ได้

การระบุเหตุการณ์ที่เกิดขึ้นเป็นครั้งคราว: Non-Stationary light curves บางครั้งมีสาเหตุจากเหตุการณ์ที่เกิดขึ้นชั่วขณะ หนึ่ง เช่น การเกิดแสงแฟลร์ของดาวฤกษ์ และเหตุการณ์ที่นานครั้งจะเกิดขึ้น เช่น ซูเปอร์โนวา การระบุเหตุการณ์เหล่านี้ช่วยให้นักดาราศาสตร์สามารถเข้าใจกระบวนการทางกายภาพของพวกมันได้

รูปที่ 1 แสดงให้เห็นถึงความแตกต่างระหว่าง Stationary light curves และ Non-stationary light curves โดยเราจะเห็นว่า Stationary light curves มีค่ากวัดแกว่งอยู่รอบๆค่าใดค่าหนึ่ง(ค่าเฉลี่ยคงที่) และมีขนาดของการกวัดแกว่งคงที่(ค่าความแปรปรวนคงที่) ในขณะที่ Non-Stationary light curves นั้นมีค่าการกวัดแกว่งที่ไม่ได้อยู่รอบค่าใดค่าหนึ่ง(ค่าเฉลี่ยคงไม่ที่) และมีขนาดของการกวัดแกว่งที่ไม่คงที่(ค่าความแปรปรวนไม่คงที่)

Autoregressive Model(AR Model)

Autoregressive Model เป็นโมเดลทางสถิติที่ใช้ในการวิเคราะห์และทำนายข้อมูลชุดตัวเลขในโดเมนเวลา โมเดลนี้ได้รับการใช้งานอย่างแพร่หลายในการศึกษาและการทำนายแนวโน้มของชุดข้อมูลที่มีการเปลี่ยนแปลงตามเวลา เช่น ราคาหุ้น, อุณหภูมิประจำวัน, หรือปริมาณการขายของสินค้า อีกทั้งมันยังสามารถใช้เป็นเครื่องมือในการสร้าง light curves ( Fourier Methods by Matteo Bachetti & Daniela Huppenkothen) วัตถุทางดาราศาสตร์บางประเภท โดยเฉพาะวัตถุที่มี light curves เป็น Stationary light curves

โดย Autoregressive Model ถูกสร้างขึ้นจากแนวคิดที่ว่าข้อมูลในปัจจุบันนั้นสามารถอธิบายได้ด้วยชุดข้อมูลในอดีตและสามารถนำมาทำนายสิ่งที่จะเกิดขึ้นในอนาคตได้อีกด้วย โดยสามารถเขียนเป็นสมการได้ดังนี้

$$ x_t = \sum_{i=1}^{p} \phi_i x_{t-i} + \varepsilon_t $$

เมื่อ :

  • \(x_t\): เส้นโค้งแสง ณ เวลา t ใดๆ
  • \(\phi_i\): Coefficients of the AR model (weight of past values) ค่าสัมประสิทธิ์ของโมเดล AR \(\phi_1\) หมายถึงน้ำหนักของค่าที่ในอดีดผ่านมาล่าสุด และ \(\phi_2\) หมายถึงน้ำหนักของค่าอดีตในอันดับสองของค่าปัจจุบัน
  • \(p\): ค่า Order ของโมเดล AR ซึ่งจะกำหนดจำนวนของค่าในอดีตที่ใช้ในการทำนายค่าปัจจุบัน แต่กรณีของเราหมายถึงจำนวนของค่าในอดีตที่ต้องการใช้สร้าง light curves ในปัจจุบัน
  • \(\varepsilon_t\): ค่าความคลาดเคลื่อนแบบสุ่ม(white noise)
โดยตัวอย่างของโค้ด AR โมเดลที่ใช้ในการการสร้าง light curves จะเป็นดังนี้ 
			
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

def generate_AR_series(n_obs, phi_values):
    """
    Generate a synthetic time series based on an AR(p) model.

    Parameters:
    - n_obs: Number of observations
    - phi_values: Coefficients for the AR(p) model (list or array)

    Returns:
    - time_series: Generated time series
    """
    p = len(phi_values)
    np.random.seed(42)

    # Generate white noise
    white_noise = np.random.normal(size=n_obs)

    # Generate time series based on AR(p) model
    time_series = [white_noise[i] for i in range(p)]

    for t in range(p, n_obs):
        X_t = sum(phi * time_series[t-i-1] for i, phi in enumerate(phi_values)) + white_noise[t]
        time_series.append(X_t)

    return time_series

Moving-Average Mode (MA model)

Moving-Average Model เป็นหนึ่งในแนวทางการวิเคราะห์ข้อมูลที่เกี่ยวข้องกับแนวโน้มของข้อมูลในโดเมนเวลา ซึ่งใช้กันทั่วไปในการประมวลผล ทางเศรษฐมิติ และทางวิทยาศาสตร์ วิศวกรรมศาสตร์ต่างๆ โมเดลเหล่านี้มีเอกลักษณ์เฉพาะด้วยการใช้ฟังก์ชันตอบสนอง(response function) ซึ่งอธิบายว่าค่าในอดีตของโดเมนเวลาใดๆทั้งหมด ส่งผลค่าปัจจุบันอย่างไร ซึ่งแตกต่างจากโมเดล AR ที่จะกำหนดว่าเราจะพิจาณาจำนวนข้อมูลในอดีตเท่าไรก่อนจะนำมาคำนวนค่าในปัจจุบัน

ในโมเดล MA นั้นค่าปัจจุบันของโดเมนเวลาจะได้มาจากการ convolving( ด้วย convolution theorem) ที่ค่าผ่านมาของโดเมนเวลาใดๆด้วยฟังก์ชันตอบสนอง เพื่อให้ได้ค่าในปัจจุบันตามสมการ :

$$ x_t = \sum_{i=-\infty}^{\infty} C_i x_{t-i} $$

เมื่อ :

  • \(x_t\): เส้นโค้งแสง ณ เวลา t ใดๆ
  • \(C_i\): ฟังก์ชันการตอบสนอง(response function)หรือค่าสัมประสิทธิ์ตัวกรอง โดยพิจารณาการมีส่วนร่วมของค่าในอดีตแต่ละค่ากับค่าปัจจุบันของโดเมนเวลาตามฟังก์ชันการตอบสนองใดๆ
  • \(x_{t-i}\): เป็นพจน์ที่แสดงถึง ค่าที่ผ่านมาของโดเมนเวลาจะขึ้นอยู่กับ \(lag k\) โดยที่ \(k\) มีค่าตั้งแต่ \(-\infty\) ไปจนถึง \(-\infty\)
โดยตัวอย่างของโค้ด MA โมเดลที่ใช้ในการการสร้าง light curves จะเป็นดังนี้ 
			
import numpy as np
import matplotlib.pyplot as plt

def generate_ma_time_series(response_function, noise_level, num_points):
    """
    Generate a time series using a Moving Average (MA) model with a response function.

    Parameters:
    - response_function: Response function (transfer function) of the MA model.
    - noise_level: Level of white noise to add.
    - num_points: Number of points in the time series.

    Returns:
    - time_series: Generated time series.
    """

    # Generate white noise
    white_noise = np.random.normal(scale=noise_level, size=num_points)

    # Compute response function for past values
    response_values = response_function(np.arange(len(white_noise)))

    # Convolve past values with the response function to get present values
    time_series = np.convolve(white_noise, response_values, mode='full')

    return time_series,  response_values
