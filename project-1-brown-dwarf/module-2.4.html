LAB-1: การจำแนกดาวแคระน้ำตาลด้วย Random Forest

ข้อมูลการสังเกตการณ์ดาวแคระน้ำตาล

การจัดการกับข้อมูลที่หายไป (Missing value)

จากข้อมูลจะเห็นได้ว่าค่า Magnitue ของแต่ล่ะ Filter ของดาวบางส่วนขาดหายไป (ค่า NaN) ดังรูปที่ 1 และหากพิจารณา Data fame 'all_df' ก่อนหน้านี้ จะพบว่าค่าดัชนีสีมีค่าเป็น NaN อยู่เช่นกัน ดังรูปที่ 2

ในการจัดการกับค่าที่หายไป เราจะใช้ Iterative Imputer จาก Library 'Scikit-learn' ในการเติมค่าที่หายไป วิธีการนี้อาศัยการทำนายค่าด้วยการวิเคราะห์แบบถดถอย (Regression) ด้วย Machine learning โดยวิเคราะห์จาก function หลายมิติที่ประกอบด้วยความสัมพันธ์ระหว่างข้อมูลของค่าในหลัก (column) อื่นๆประกอบการทำนาย

ขั้นแรกของการทำ Iterative Imputer จะเริ่มจากการสร้าง Data frame 2 ตัว คือ df_imp_tr และ df_imp_eva สำหรับนำไปฝึกฝน (train) และประเมินผล (evaluate) แบบจำลองการเรียนรู้ของเครื่อง โดยอิงจากข้อมูล all_df เดิม และตัดข้อมูลแถว (row) ที่เป็น 'NaN' ออก 

			
#ทำการกำจัด ข้อมูลในแถว(row)ที่มี "NaN" ออกไปทั้งหมด ด้วยคำสั่ง '.dropna()'
df_imp_tr = all_df.dropna()
df_imp_eva = all_df.dropna()
#ทำการตัด column ของ label class ออก เนื่องจากไม่มีความจำเป็นขั้นตอนนี้
df_imp_tr = df_imp_tr.drop('label', axis = 1)
df_imp_eva = df_imp_eva.drop('label', axis = 1)
display(df_imp_eva)
display(df_imp_tr)

เมื่อให้แสดงผลข้อมูล ทั้ง df_imp_tr และ df_imp_eva ยังคงมีข้อมูลที่เหมือนกันทุกประการ ดังนี้

จากนั้นจะทำการสุ่มข้อมูล 5 % ใน df_imp_tr ให้เป็น 'NaN' value เพื่อใช้ในการฝึกฝน Iterative Imputer ของเรา

**เหตุใดเราจึงต้องตัด 'NaN' value ใน all_df ก่อน แล้วจึงไปสุ่ม 5% ให้เป็น NaN อีกรอบก่อนนำไปฝึกฝน Iterative Imputer ของเรา ทำไมไม่ใช่ all_df ที่มี 'NaN' value อยู่แต่เดิมแล้วในการฝึกฝน

เหตุผลคือ 'NaN' value ใน all_df ของเรานั้นเราไม่รู้ค่าแท้จริงของมัน ถ้าหากเราใช้มันในการฝึกฝน Iterative Imputer เราไม่อาจรู้ได้เลยว่า Iterative Imputer ที่ได้ออกมานั้นมีประสิทธิภาพเป็นอย่างไร แต่ถ้าใช้ df_imp_tr ที่ถูกสุ่มให้ข้อมูลบางส่วนเป็น 'NaN' value ก่อนนำไปฝึกฝนนั้นและเมื่อผ่านการ Imputation (การทำนายค่า 'NaN' value เป็นค่าจริงๆ) แล้ว เราสามารถใช้ df_imp_eva ซึ่งเหมือนกัยทุกประการกับ df_imp_tr ก่อนที่ถูกสุ่มค่า ในการประเมิณผลได้ค่าที่ผ่านการทำนายได้

โดยตัวอย่างโค้ดจะเป็นดังนี้ 

			
# สร้าง array ของ True, False ที่มี shape ตรงกับ df_imp_tr ทุกประการ โดยสุ่มให้มีค่า True 5%(0.05) และค่า False 95%(0.95) 
missing_mask = np.random.choice([True, False], size=df_imp_tr.shape, p=[0.05, 0.95])
# ให้ค่าในตำแหน่งของข้อมูลใน df_imp_tr ซึ่งตรงกับ missing_mask ที่มีค่าเป็น True เท่ากับ 'NaN' value ซึ่งจะทำให้ค่าใน df_imp_tr มีค่าเท่ากับ 'NaN' value  5% พอดี
df_imp_tr[missing_mask] = np.nan
display(df_imp_tr)

ผลลัพธ์จะเป็นดังนี้

ต่อไปเป็นขั้นตอนการฝึกฝน IterativeImputer โดย เราจะกำหนดให้ IterativeImputer ของเรานั้น ใช้ Extra Trees Regressor ด้วย hyperparameter คือ n_estimators = 150, max_features = 14, max_depth=15, min_samples_split =12, initial_strategy = 'median' และมีการวนซ้ำใน IterativeImputer มากที่สุด 20 ครั้ง (max_iter = 20) ในการใช้งานงานในจริงๆ parameter เหล่านี้ควรจะต้องมีการ fine-ture เพื่อให้ได้ IterativeImputer ที่ดีที่สุด แต่เนื่องจากเราทำแบบฝึกหัดตามขั้นตอนใน Avdeeva 2023 ดังนั้นเราจะใช้ parameter ต่างๆ ตามบทความนี้ไปเลย โดยความหมายของ parameter ต่างๆ สามารถอ่านเพิ่มเติมได้ใน Extra Trees Regressor และตัวอย่างการ fine-ture hyperparameter สามารถอ่านเพิ่มเติมได้ใน Iterative Imputation

และในขั้นตอนการประเมิณผลของ IterativeImputer ของเราจะใช้คำสั้ง '.isclose' ใน Library numpy โดยเป็นการเปรียบเทียบค่าที่ถูก Imputation กับค่าแท้จริงนั้นใกล้เคียงกันหรือไม่ จากนั้นใช้คำสั้ง '.mean' คำนวนความแม่นยำ

โดยตัวอย่างโค้ดจะเป็นดังนี้ 

			
from sklearn.experimental import enable_iterative_imputer
from sklearn.impute import IterativeImputer
from sklearn.ensemble import ExtraTreesRegressor

#ตั้งค่า Parameter ต่างๆใน IterativeImputer
imp = IterativeImputer(estimator=ExtraTreesRegressor(n_estimators =150, max_features = 14, max_depth=15,
                                                                           min_samples_split =12), initial_strategy = 'median', max_iter =20)
#การฝึกฝน IterativeImpute ที่เราสร้าางขึ้นด้วยข้อมูล "df_imp_tr" ด้วยคำสั่ง ".fit"
imp.fit(df_imp_tr)

# ทำการ Imputation ชุดข้อมูลของเราด้วย imputing machince learning ทีถูกฝึกฝนมาแล้ว ด้วยคำสั่ง '.transform'
Im_data = pd.DataFrame(imp.transform(df_imp_tr))


# คำนวนความแม่นยำของIterativeImputerของเรา
accuracy = np.mean(np.isclose(df_imp_tr[missing_mask], Im_data[missing_mask], equal_nan=True))
print("ค่าความแม่ยำของการ Imputation :", accuracy*100)
display(Im_data)

ผลลัพธ์จะเป็นดังนี้

ซึ่งเราจะเห็นว่าค่าความแม่นยำในการ Imputation อยูที่ ~78% และค่า 'NaN' value ใน Data frame นั้นถูกทำให้เป็นค่าตัวเลขจริงๆทั้งหมดแล้วหากเปรียบเทียบกับ Data frame ในขั้นตอนก่อนหน้า
