LAB-1: การจำแนกดาวแคระน้ำตาลด้วย Random Forest

ข้อมูลการสังเกตการณ์ดาวแคระน้ำตาล

การอ่านไฟล์ข้อมูลจาก Directory ด้วย Pandas และการคัดเลือกข้อมูลที่จำเป็น

Pandas เป็น Library ของ Python ที่เหมาะกับการใช้จัดการและวิเคราะห์ข้อมูลที่อยู่ในรูปแบบของตาราง โดยเราสามารถโหลดข้อมูลตารางไฟล์สกุลต่างๆ เช่น .xlsx และ .csv และทำการจัดเก็บข้อมูลด้วย Data frame ซึ่งมีการแบ่งข้อมูลเป็นแถว (Row) กับคอลลัมบ์ (Column)
