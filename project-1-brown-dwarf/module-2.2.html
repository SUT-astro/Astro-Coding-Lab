LAB-1: การจำแนกดาวแคระน้ำตาลด้วย Random Forest

ข้อมูลการสังเกตการณ์ดาวแคระน้ำตาล

การอ่านไฟล์ข้อมูลจาก Directory ด้วย pandas และ extract ข้อมูลที่จำเป็น

pandas เป็น libraly ของ python ที่เหมาะกับการใช้ในการจัดการและวิเคราะห์ข้อมูลที่อยู่ในรูปแบบของตาราง โดยจะจัดเก็บข้อมูลด้วย Data frame ซึ่งมีการแบ่งข้อมูลเป็น Row กับ Column เราสามารถโหลดข้อมูลจากไฟล์สกุลต่าง ๆ ได้ เช่น .xlsx (ไฟล์ Excel), .csv, และสกุลอื่น ๆ

ในแบบฝึกหัดนี้ เราจะทำการดึงไฟล์ข้อมูลของเราจาก google drive โดยใช้คำสั่ง pd.read_csv โดยตัวอย่างของโค้ดจะเป็นดังนี้ 

			
import pandas as pd
# file_id ของไฟล์ใน google drive
file_id = '1hQYrh4zpfxYu-soF4O-rXA8YF_8J-JQc'
# โหลดไฟล์จาก google drive ด้วย link
data = pd.read_csv(f'https://drive.google.com/uc?id={file_id}')

ผลที่ได้จะเป็นดังนี้

เมื่อเราได้ข้อมูลมาแล้วแล้ว ต่อมาเป็นขั้นตอนการ Extract เฉพาะข้อมูลที่เราต้องการใช้ โดยการไม่เรียกใช้ข้อมูลที่ไม่จำเป็น เช่น เราไม่ต้องการส่วนของ number_tag และ feature_4 เหลือไว้เพียง แค่ส่วนของ feature_1, e_feature_1, feature_2, e_feature_2, feature_3, e_feature_3, 'label' ซึ่งตัวอย่างจะถูกแสดงใน code ด้านล่าง 

			
# Extract the data from the dataset
df = data[['feature_1', 'e_feature_1', 'feature_2', 'e_feature_2', 'feature_3', 'e_feature_3',  'label']]

ผลที่ได้จะเป็นดังนี้

ต่อมาเป็นการตรวจสอบมูลชุดข้อในเบื้องต้น จากข้อมูลตาราง Dataframe ด้านบน เราจะเห็นว่ามี label ของข้อมูลอยู่ 2 ชนิดคือ

postive class คือดาวแคระน้ำตาล (label เป็น 1)

negative class คือไม่เป็นดาวแคระน้ำตาล (label เป็น 0)

โดยเราใช้คำสั้ง df.loc ใช้การ sorting มูลในส่วนที่เราสนใจ เช่น ต้องการ sorting possitive class และ negative class เพื่อดูจำนวนข้อมูลในแต่ class ซึ่งตัวอย่างโค้ดจะถูกแสดงด้านล่าง 

			
##เลือกทุกแถวที่มี label = 1 ลงใน Pos_class
Pos_class=df.loc[df['label'] ==1 ]
แสดง Dataframe ที่ชื่อ "Pos_class"
display(Pos_class)


##เลือกทุกแถวที่มี label = 0 ลงใน Neg_class
Neg_class=df.loc[df['label'] ==0 ]
แสดง Dataframe ที่ชื่อ "Neg_class"
display(Neg_class)

ผลที่ได้จะเป็นดังนี้

จะเห็นว่า negative class มีมากกว่า possitive class ประมาณ 2.5 เท่า ดังนั้น จึงต้องมีการทำให้ข้อมูลสมดุลกันด้วยการ Augmention(การเติม)ข้อมูล ซึ่งจะแสดงใน section ถัดไป

และในส่วนของแบบฝึกหัดของนักศึกษาจะใช้ file id คือ '1RPTYy1D0Z9mDiYFy_ABHClOcLCZnv-zB' และต้องการ extract ข้อมูลในส่วนของ column :i, e_i, z, e_z, y, e_y, W1, e_W1, W2, e_W2, J, e_J, H, e_H, K, e_K, label
